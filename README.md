# 리눅스 프로세스 관련 명령어 정리 (top, ps, jobs, kill)

## 📌 1. top

### 1-1. top 이란?
`top` 명령어는 시스템에서 실행 중인 프로세스를 **실시간으로 모니터링할 수 있는 명령어**이다.  
CPU/메모리 사용량을 확인하거나 현재 시스템 상태를 파악할 때 사용한다.

### 1-2. 자주 쓰는 옵션
| **옵션** | **설명** | **예시** |
|------|------|-----------|
| -u 사용자명 | 특정 사용자 프로세스만 보기 | `top -u ubuntu` |
| -p PID | 특정 PID만 보기 | `top -p 1234` |
| h | 도움말 | 실행 중 h 누르기 |

### 1-3. 사용 예시
```bash
top
top -u root
```

## 📌 2. **ps**

### 2-1. **ps란?**

ps(process status)는 **현재 실행 중인 프로세스 목록을 출력하는 명령어**이다.
top과 달리 화면이 실시간 갱신되지 않고, 명령을 실행한 시점의 정보만 출력한다.


### 2-2. **자주 쓰는 옵션**

| **옵션** | **설명** |  **예시** |
|------|------|-----------|
| **aux** | 모든 사용자 프로세스 상세 출력 | `ps aux` |
| **-ef** | 표준 형식으로 전체 프로세스 출력 | `ps -ef` |
| **-e** | 모든 프로세스 출력 | `ps -e` |
| **-u 사용자명** | 특정 사용자 프로세스만 출력 | `ps -u ubuntu` |
| **-p PID** | 특정 프로세스만 출력 | `ps -p 1234` |


### 2-3. **사용 예시**

```bash
ps aux
ps -ef
ps -u ubuntu
ps -p 1234
```

## 📌 3. **jobs**

### 3-1. **jobs란?**
jobs 명령어는 현재 셸에서 실행 중이거나 중지 상태인 **백그라운드 작업 목록**을 보여준다.
fg, bg 명령어와 함께 사용된다.

### 3-2. **자주 쓰는 옵션**

| **옵션** | **설명** | **예시** |
|------|------|-----------|
| **jobs** | 기본 작업 목록 출력 | `jobs` |
| **-l** | PID 포함해서 출력 | `jobs -l` |
| **-r** | 실행 중인 작업만 출력 | `jobs -r` |
| **-s** | 멈춘 작업만 출력 | `jobs -s` |

### 3-3. **사용 예시**

```bash
jobs
jobs -l
jobs -r
jobs -s
```
---
## 📌 4. **kill**

### 4-1. **kill 이란?**
kill 명령어는 특정 프로세스에 **신호(signal)를 보내 종료하거나 제어**할 때 사용한다.
기본 종료 신호는 SIGTERM(15)이며, 강제 종료는 SIGKILL(9)이다.

### 4-2. **주요 신호(Signal)**

| **번호** | **이름** | **설명** | **예시** |
|------|------|------|------|
| **15** | SIGTERM | 정상 종료 요청 (기본) | `kill 1234` |
| **9** | SIGKILL | 강제 종료 | `kill -9 1234` |
| **1** | SIGHUP | 설정 다시 읽기 | `kill -1 1234` |
| **2** | SIGINT | Ctrl+C와 동일 | `kill -2 1234` |

### 4-3. **사용 예시**

```bash
kill 1234
kill -9 1234
kill -1 1234
kill -2 1234
```
